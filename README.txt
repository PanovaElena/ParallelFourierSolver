To build the project use cmake. If your OS is Windows, move file 
lib/fftw-3.3.5_windows_x64/libfftw3-3.dll to directories with executable files.

For easy launch use ./files/[example]/run/run_all_and_plot_graphics_windows.py.
Command arguments can be set in ./files/[example]/run/args.py

The executable files can be find in directories:
build/src/Examples/RunningWave/SequentialFieldSolver_RunningWave
build/src/Examples/RunningWave/ParallelFieldSolver_RunningWave
build/src/Examples/SphericalWave/SequentialFieldSolver_SphericalWave
build/src/Examples/SphericalWave/ParallelFieldSolver_SphericalWave

To get information about command arguments use option --help.
